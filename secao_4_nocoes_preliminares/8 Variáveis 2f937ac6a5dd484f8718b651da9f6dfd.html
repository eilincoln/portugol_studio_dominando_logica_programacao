8. Variáveis

Mas o que é uma variável??

É uma variável nada mais do que um objeto ou entidade armazenada na memória do computador, que representa um valor ou uma expressão.

Chama-se variável pois p valor contido nesse espaço de memória do computador varia com o tempo, não é um valor fixo

Quanto uma variável é declarada em um programa, na verdade está definindo e reservando um espaço na memória para armazenar o valor que aquela variável conterá em determinado tempo de execução do programa.

Nos algoritmos criados para realizar tarefas de computação utilizamos variáveis para manipular dados, por exemplo:

  • Nome
  • Idade
  • Altura
  • peso
  • data de nascimento
  • sexo
  • saldo
  • e mais

Para otimizar os dados da memória, é armazenado somente um dado na memória.

Memória

Meio físico para armazenar dados temporariamente ou permanente (TANEBAUM, 1997, P.212)

Para simplificar, pense na memória como um armário de roupas com várias gavetas. Cada gaveta representa uma variável, e nela são guardados tipos de dados.

Cada gaveta armazena apenas um tipo de dado, no nosso caso, roupas, que poderiam ser:

  • Calças
  • Camisas
  • Meias
  • Bermudas e mais

Logo teríamos uma gaveta apenas para cada tipo de roupas. Apenas para cada tipo de roupa e na gaveta será armazenado só aquele tipo de roupa.

Pense assim:

  • A roupa seria o tipo da variável
  • A gaveta seria a variável que armazena os dados
  • E o armário seria a memória do computador

Antes de continuar

Precisamos ter em mente o que é um dado, e qual seu conceito.

Um dado em sí, não possui significado relevante e também não conduz a nenhuma compreensão, representando algo que não possui sentido a princípio.

Portanto, não tem valor, para poder gerar alguma informação.

Já a informação é a organização dos dados. De forma que vai passar alguma informação.

A informação é o conjunto e a consolidação dos dados de forma a fundamentar o conhecimento. Sendo assim a informação possui valor agregado.

Níveis de Abstração

Com estes conceitos definidos e aproximando-se da maneira como os computadores manipulam as informações, é possível dividi-las em tipos primitivos, que serão os tipos básicos que utilizaremos para trabalhar com algoritmos no Portugol Studio.

Tipo Caractere

É o formato que sempre recebe os caracteres alfanuméricos. E deve sempre estar delimitado por apóstrofes.

Tipo Cadeia

Representa uma sequência de caracteres que pode ser constituída de letras, números ou símbolos. O tipo cadeia deve ser acompanhado de aspas dublas.

Tipo lógico

Armazena os valores lógicos True e False

Tipo Vazio

Vazio é usado para um resultado que retorna algo, mas que não fornece nenhum resultado.

É um tipo mais complexo de se entender, mas é usado mais para quando estiver fazendo algo que não precisa retornar algo, mas vai retornar.

Constante

Como já discutido anteriormente, o valor de uma variável pode ser modificado ao longo de uma execução de um algoritmo. Mas podem ocorrer casos em que é necessário armazenar valores que não sofram quaisquer alterações ao longo da execução do algoritmo. Neste contexto, surgem as constantes.

AS constantes são criadas conforme as regras para criação das variáveis, mas diferentes pelo fato de não alterarem o valor à elas atribuídos, além disso geralmente possuem um escopo global, sendo possível fazer o chamado a partir de qualquer ponto do algoritmo.
